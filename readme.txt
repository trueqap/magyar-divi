DIVI magyar kiegészítések és fordítás 


== Description ==

Magyar Divi beállítások (fordítás, megjelenés és egyéb fejlesztések)


== Installation ==

1. Töltsd le a bővítményt
2. Wordpress-ben bővítmények / új hozzáadása menüben fel kell tölteni
3. Működik

== Frequently Asked Questions ==

= Miben tér el a fordítás az eredeti DIVI-től? =

A DIVI fordítása sok Google fordítást tartalmaz, ami sok esetben érthetetlen és magyartalan. A bővítmény ezeket a hibákat igyekszik orvosolni.
